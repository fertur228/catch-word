# TakeWord — мастер-документ проекта

> **TakeWord** (англ. *catch* + *word* — «лови слово»). Учи язык, наводя камеру на реальный мир: навёл на предмет → приложение узнаёт его, вырезает в стикер и даёт слово, перевод, произношение и живой пример. Слово копится в личную коллекцию и всплывает на повторение.
>
> Tagline: *See it. Catch it. Speak it.*

Это единый источник правды по проекту: держи в корне репозитория, давай Claude Code как контекст. Помечено `[v1]` (делаем сразу) и `[later]` (слой позже).

---

## 0. TL;DR
- **Что:** фотословарь. CapWords-клон по механике, но бьём по его слабым местам.
- **Платформа:** iOS, амер рынок. Возраст 4+.
- **Стек:** Expo (React Native) + Supabase (бэкенд-прокси) + RevenueCat (подписки) + облачная vision-модель.
- **Деньги:** freemium, 3 тарифа (Free / Basic / Premium), цена чуть выше CapWords.
- **Главный риск:** не продукт и не цена — **дистрибуция без фичеринга**. Это и есть настоящая работа.
- **Реалистичный потолок:** даже награждённый CapWords делает ~$20k/мес. Это «крепкий инди», не единорог. Под инкубатор и обучение пайплайну — отлично.

## 1. Концепт и петля
Навёл камеру на предмет → снял кадр → **подтверждение** (вырезка фона + распознавание через бэкенд) → сохранилась **карточка** (стикер + слово + перевод + произношение + пример) → **коллекция** растёт → **повтор** по spaced repetition. Лимит сканов и набор фич зависят от тарифа.

Эмоциональный крючок: «весь мир — твой словарь», коллекционирование стикеров, обучение из реальной жизни, а не из списков.

---

## 2. Позиционирование: «то же самое, только не бесит»
CapWords — лауреат Apple Design Award 2025, но в отзывах его ругают за конкретные вещи. Это наш список, где мы выигрываем:

| Слабость CapWords | Наше решение |
|---|---|
| Перевод дословный, баги в примерах | Хороший промпт + сверка, живые примеры |
| Робо-голос озвучки | Системный голос (`expo-speech`), сразу естественнее |
| «1 скан и стена» + списания без напоминания | Щедрый бесплатный тест + честное напоминание перед списанием |
| Лаги, сдвиг озвучки, баги категорий | Делаем стабильно и надёжно |

Отстройка — не «дешевле», а **качество + честность + надёжность**.

## 3. Конкуренты (трезво)
- **CapWords** — оригинал. ~500k загрузок, ~$20k/мес, 10 языков. Его двигатель роста — награда + App of the Year = бесплатные установки (CAC ≈ $0). Награду скопировать нельзя.
- **Прямые клоны** — SnapWords, StickerWords, ScanWords. Модель один-в-один (free + Pro), но БЕЗ награды → дерутся на чистом ASO/органике. Новые появляются ежемесячно.
- **Косвенные** — Google Lens/Translate (бесплатный гигант, без петли обучения), Duolingo/Babbel/Memrise (структурные курсы, другая лига).

Вывод: полоса дженерик-вокабуляра забивается. Наш моат = качество + честный пейволл + **дистрибуционный крючок** (см. §10). Быть клоном №5 без крючка — слабо.

## 5. Экраны `[v1]`, если не помечено иначе

### 5.1 Онбординг и выбор языка
Короткий онбординг (1–3 экрана) → «Какой язык хочешь учить?» со списком флагов. При первом запуске — анонимный вход (Supabase), сохранить родной язык + язык изучения. *Тарифы: Free — 1 язык; Basic — 1–2; Premium — все.*

### 5.2 Главная / Камера
Живое превью, большая кнопка съёмки, доступ к Коллекции и Настройкам. Показывать остаток бесплатных сканов (Free). Перед съёмкой — проверка лимита; исчерпан → Пейволл.

### 5.3 Подтверждение и результат (сердце «магии»)
Вырезанный стикер на чистом фоне + слово на изучаемом языке + перевод + 🔊 + пример. Кнопки: ✓ сохранить, ↻ переснять, ✕ отменить. Шаг подтверждения специально даёт время на вызов API. Поток: захват → (вырезка фона) → POST `/recognize` → результат → по ✓ сохранить локально. 🔊 = `expo-speech`. *В `[v1]` можно без вырезки (просто фото/кроп); вырезка — `[later]`. Premium — 2–3 примера + грамматика; Free/Basic — 1 пример.*

### 5.4 Карточка слова
Стикер, слово, перевод, кнопка озвучки, пример(ы), (грамматика/род — Premium). Редактирование, удаление. `[later]`: тап по слову в примере → его значение.

### 5.5 Коллекция / Скрапбук
Сетка стикеров-карточек, прогресс, фильтр по категориям/языку. Читается из локальной БД, работает офлайн. Тап → Карточка. *Сохранённые карточки и просмотр — бесплатно навсегда (нулевая стоимость API, держит привычку).*

### 5.6 Повтор (Spaced Repetition)
Сессия флеш-карточек (слово/стикер → вспомнить → оценить). Простой SRS (SM-2-lite) + push-напоминания. *Базовый — Basic+; умный подбор — Premium.*

### 5.7 Пейволл
Три тарифа (§8), герой — годовой Premium «Best Value», 7-дневный триал, напоминание ВКЛ честно. RevenueCat: покупка, триал, восстановление, entitlement. Триггеры: лимит сканов исчерпан / попытка открыть фичу выше тарифа.

### 5.8 Настройки / Профиль
Язык изучения/родной, выбор акцента (системные голоса), управление подпиской, восстановить покупки, ссылки Privacy/Terms.

## 6. Технический стек
- **Приложение:** Expo (React Native) + TypeScript, навигация `expo-router`.
- **Среда:** VS Code + Claude Code; сборка — Xcode в фоне; ежедневно — **development build** на айфоне (не Expo Go, т.к. нужен нативный модуль).
- **Камера:** `expo-camera`. **Озвучка:** `expo-speech`. **Локальная БД:** `expo-sqlite`.
- **Вырезание фона `[later]`:** нативный модуль на Swift через `VisionKit` (`VNGenerateForegroundInstanceMaskRequest`). Тестируется только на устройстве.
- **Подписки:** RevenueCat (`react-native-purchases`).
- **Бэкенд:** Supabase — Postgres + Auth (анонимный вход) + Edge Functions (прокси) + Storage.
- **AI:** облачная vision-модель, вызывается ТОЛЬКО с бэкенда (ключ в секретах Edge Function).

## 7. Архитектура и данные

### 7.1 Поток одного скана
iPhone (Expo) → [захват, локальная вырезка фона] → отправляет {фото, язык, токен} → **твой бэкенд (Supabase Edge Function)** → [auth → проверка тарифа/лимита → кеш] → при промахе **vision-модель** → [записать в кеш] → вернуть {слово, перевод, IPA, примеры} → приложение показывает карточку, `expo-speech` озвучивает, сохраняет в локальную коллекцию.

**Ключевое:** ключ AI живёт ТОЛЬКО на бэкенде, никогда в приложении.

### 7.2 Модель данных
**Локально (`expo-sqlite`):**
- `user_profile`: id (анон), native_lang, learning_lang, tier, scans_used, scans_period_start, revenuecat_id
- `word_card`: id, image_uri, object_label, translation, pronunciation_ipa, example_sentences (json), category, created_at, srs_due_at, srs_interval, srs_ease

**На сервере (Supabase Postgres):**
- `recognition_cache`: object_key (нормализованный лейбл/хэш), lang_pair, result_json, created_at — отдаёт повторы за ~$0
- `usage`: user_id, period_start, scans_count — серверное подтверждение лимитов

### 7.3 Эндпоинт `/recognize` (Edge Function)
1. Проверить токен (Supabase Auth).
2. Проверить тариф и лимит (RevenueCat entitlement + `usage`). Превышен → 402/403.
3. Нормализовать вход, заглянуть в `recognition_cache`. Хит → вернуть мгновенно.
4. Промах → вызвать vision-модель (ключ из секретов): {label, translation, ipa, examples[]}.
5. Записать в кеш, инкрементировать usage, вернуть.
Rate-limit (~15/мин) и дневной софт-кап — здесь же.

| | Free | Basic | Premium |

- Локализовать вниз для бедных рынков (KZ ~$12–15/год).

## 9. Юнит-экономика (из расчётов)

**Допущения:** себестоимость **$0.002/скан** (средняя модель + лёгкий кеш; дешёвой ~$0.0004, кешем ~$0.001, топ-модель ~$0.009). Apple берёт **15%** (Small Business Program).

- **Net с Premium-годовой:** $39.99 × 0.85 / 12 ≈ **$2.83/мес**.
- **Точка безубытка:** ~700–1,400 сканов/мес. Обычный юзер сканит **5–30/мес** → маржа 95%+.
- **Basic закрыт хард-капом 150/мес** → в минус физически не уйдёшь.
- **На 100 установок** (конверсия ~2%): ~$51 выручки − ~$4.5 API ≈ **$46 валовой маржи**.
- **LTV установки ≈ $0.50** → платная реклама почти не окупается → рост обязан быть органическим/виральным/через фичеринг.

**Защита от абьюза/утечки (must до релиза):** ключ только на бэкенде → rate-limit ~15/мин → дневной софт-кап ~300–500 (троттлинг, не блок) → месячный софт-кап ~1000 на Premium → те же лимиты на триал → кеш частых пар.

## 10. Дистрибуция (настоящая работа)
Это узкое место, не продукт. Денежный потолок упирается сюда.

- **TikTok/Reels — «формат-машина».** Один узкий шерящийся формат, постить ежедневно. Идеи под нас: «навёл → стикер» reveal-клипы; «учим язык через предметы» (GRWM-language гибрид — живой тренд); «угадай слово». Наши стикеры сами по себе скриншот-friendly — это виральный актив.
- **ASO:** ключи «learn [язык] by photo», «photo vocabulary», «snap and learn».
- **Фичеринг Apple** — главный приз (двигатель CapWords). Полированный дизайн + история основателя из Казахстана + Apple-фреймворки → шанс на Editor's Choice.
- **Сообщества:** родительские, школы, языковые.
- Реклама почти не окупается (LTV ~$0.50) → ставка на органику. Контент-движок обязателен, не опционален.

## 11. План сборки (по слоям, под инкубатор)
1. **Скелет `[v1]`** — Expo + `expo-router`, экраны на моках, навигация, локальная БД. Симулятор. → кликабельная петля за вечер.
2. **Пейволл + RevenueCat** — экран тарифов, sandbox-покупки, entitlement.
3. **Бэкенд** — Supabase: `/recognize` с проверкой тарифа и кешем, вызывает vision-модель. Приложение зовёт его (пока без вырезки).
4. **Камера на устройстве** — `expo-camera`, перейти на development build, гонять на айфоне: съёмка → `/recognize` → карточка.
5. **Вырезка фона (натив) `[later]`** — Swift-модуль `VisionKit`, мгновенный стикер.
6. **Полировка** — SRS, настройки, акценты, напоминания.
7. **Прод** — production build и публикация через аккаунт инкубатора (их EAS / App Store Connect).

## 12. Риски
- **Дистрибуция без награды** — главный. Митигейшн: контент-движок в TikTok + заявка на фичеринг.
- **Моментальность вырезки** — нативный `VisionKit`, прощупать рано.
- **Apple Review** — придирчив к платным AI-аппкам и детской аудитории (4+). Честный пейволл и дисклеймеры снижают риск.
- **Забитость клонами** — отстройка качеством/честностью/крючком; при пробуксовке — рассмотреть более узкий угол.
