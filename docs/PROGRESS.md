# CatchWord — состояние проекта и дорожная карта

> Обновлено: 2026-06-27. Источник правды по продукту — `../catchword_project_doc.md` (спека).
> Этот файл — что уже сделано, как устроено и что дальше.

## TL;DR
Рабочее iOS-приложение на **Expo SDK 56 + TypeScript + expo-router**, доведённое до уровня аналога **CapWords**, но **полностью на моках** (нет бэкенда, реального распознавания и оплаты). Запускается на симуляторе; на реальный iPhone — после ручной настройки подписи в Xcode (см. `SETUP.md`).

## Стек
- Expo SDK **56** (React Native 0.85, React 19), TypeScript (strict), expo-router (роуты в `src/app`).
- Эксперименты включены: `typedRoutes`, `reactCompiler`.
- Нативные модули: `expo-camera`, `expo-speech`, `expo-sqlite`, `expo-dev-client`, `expo-symbols` (SF Symbols).
- Анимации: `react-native-reanimated` v4.
- bundle id: `com.almazbukayev.catchword`.

## Что готово (всё на моках)
- **Онбординг** + выбор изучаемого/родного языка (флаги).
- **3 вкладки**: 📷 Камера · 🗂 Коллекция · 🔁 Повторение.
- **Камера** — корректный запрос разрешения, сканирующая рамка, кнопка съёмки, счётчик сканов → пейволл.
- **Распознаю…** — переходный экран-анимация между съёмкой и результатом.
- **Результат** — reveal-анимация; слово редактируется, автоподсказки из словаря, авто-перевод+транскрипция; Сохранить/Переснять/Отмена.
- **Коллекция/скрапбук** — статистика (поймано/выучено/серия), поиск, чипы-категории, **переключатель Сетка / По датам** (Сегодня/Вчера/число).
- **Карточка слова** — стикер, IPA, 🔊, примеры, мастерство, удаление.
- **Повторение (SRS)** — выбор режима: **флеш-карточки** или **тест** (варианты ответа), прогресс, итог сессии.
- **Пейволл** — тарифы ровно из спеки §8 (Free/Basic/Premium/Lifetime, 7-дн триал, годовой Best Value), переключатель Месяц/Год, честное напоминание. Кнопки оплаты — заглушки.
- **Настройки** — смена языков, голос/акцент, подписка, о приложении.
- Локальная **БД** (expo-sqlite) с безопасной миграцией схемы.

## Как запустить (кратко; подробно — `SETUP.md`)
```bash

npx expo run:ios            # первый раз: собрать + запустить на симуляторе
```
